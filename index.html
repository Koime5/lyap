On design

Simplicity, scaled down

Lyap takes its instincts from Alpine, HTMX, and Svelte — declarative markup, server-friendly interaction, fine-grained reactivity — and rebuilds them in its own syntax, tuned for one purpose: staying out of the way.

I.

Lightweight

No virtual DOM overhead, no complex bundler setup. Drop in a single script tag and start writing dynamic templates immediately.

II.

Scalable

Begin with a touch of interactivity on a static page. Grow into reactive stores, components, and AJAX swaps without a rewrite in between.

III.

Explicit Syntax

A concise set of @ and : shorthands designed to prevent ambiguity and stay completely readable across any editor font.

A first look

State and behavior, in the markup

Designed with the same regard for legibility that drew us to modern web primitives in the first place.

counter.html 
<!-- state lives where you use it -->
<div ly-data="{ count: 0 }">

  <button @click="count++">
    +1
  </button>

  <span :text="count">0</span>

</div>
